# Career Compass

I built this to help people figure out what their next move should be career-wise. Not everyone knows what path to take — whether it's a promotion, a switch to a different role, or just knowing what skills matter in your field.

**What it does:** You set your current role and your goal. It shows you possible career paths, what skills you need, and tracks your progress. Gives you a roadmap instead of just wandering around guessing.

## Tech

- React, TypeScript
- Tailwind CSS, shadcn/ui
- Vite, React Query
- Recharts for visualizations

## Setup

```bash
git clone https://github.com/salim1117/career-compass.git
cd career-compass
npm install
npm run dev
```

Runs at `http://localhost:5173`.

## How it works

1. Create a profile with your current role
2. Pick your target role or goal
3. See the career paths between them
4. Identify skills you need vs. what you have
5. Track progress as you learn

Built for people who want clarity on their next step without paying a career coach.
